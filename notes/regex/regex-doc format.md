# format

regex is a pattern-matching language

many programming languages and system tools
have a regex engine

---
## system tools with regex

* grep
* sed
* perl
* vim
* less

---
## common regex format

* `/PATTERN/` - match
* `s/PATTERN/REP/` - replace

You can also have FLAGS at the end:

* `/PATTERN/FLAGS`
* `s/PATTERN/REP/FLAGS`
